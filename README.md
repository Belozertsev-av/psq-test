### Установка и запуск

1) Склонируйте git-репозиторий 
2) Подгрузите зависимости
```
npm install
```
3) Запустите в терминале лже-api
```
npm run api
```
4) Запустите еще один терминал и запустите приложение
```
npm run dev
```
5) Приложение будет доступно на локальном хосте на 5174 порту.
http://localhost:5174/
